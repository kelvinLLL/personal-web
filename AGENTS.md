# AGENTS

This repo uses docs-first feature development.

Before feature implementation:

1. Use the repo-local skill at `.agents/skills/sdd-feature-development/SKILL.md`.
2. Read `docs/development-rules.md`.
3. Open or create the relevant feature doc described by `docs/features/README.md`.
4. Update the feature doc before writing code.

No code before doc.

Use `docs/development-rules.md` for durable project rules and `docs/features/<feature>.md` for feature-local context.
